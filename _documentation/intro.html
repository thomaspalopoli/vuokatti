Overview

Make is a responsive and multipurpose admin and frontend theme powered with Twitter Bootstrap Framework 3. Make can be used for any type of web applications: custom admin panels, admin dashboards, CMS, CRM, SAAS and websites: business, corporate, portfolio, blog. Make has a sleek, clean and intuitive flat style design which makes your next project look awesome and yet user friendly. Make has a huge collection of plugins and UI components and works seamlessly on all major web browsers, tablets and phones.

We put a lot of love and effort to make Make a useful tool for everyone.

We are keen to release continuous long term updates and many of new features will be coming soon in the future releases. Once you purchased Make, you will be entitled to free download of all future updates for the same license.

Styles

Heads up! We use 3 CSS files in this theme: a style file, ui file and theme file.

Stylesheets

Here's a list of the stylesheet files we are using with this template, you can find more information opening each file:

File Name Description
style.css The main CSS file of the template. You will find all general style for sidebar, topbar and page content.
ui.css This file content all user interface style, such as button, form, tables...
gray.css Theme file define all color variations. There are 5 main colors:blue, gray, dark, orange and red.

Javascript

Here's a list of the script files we are using with this template, you can find more information opening each file.

Sometimes specific pages need specific script files. Pease see demo pages and / or plugins page too see which scripts need to be included: plugins.html.

File Name Description
jquery-1.11.1.min.js Main Jquery file.
bootstrap.min.js Bootstrap JS file.
retina.min.js Script to adapt image for retina screen.
select2.min.js Custom select input.
icheck.min.js Custom checkbox & radio inputs.
backstretch.min.js Background image and image slider.
application.js Manage application script.
