Customize to Fit your Needs

Style

To overide the theme CSS styles you can use common-files/css/custom.css for your own customization. It will be easier for your futur updates if you keep your own modifications in a separate file.

Don't forget to add custom.css in the head of your page to apply your custom CSS.

Javascript

To add javascript functions, you can use common-files/js/custom.js for your own customization. This will make the future updates easier if you keep your own Javascript code seperate.

Don't forget to add custom.js in the footer of your page to apply your custom Javascript.

Create new page

To create a new page, you can use an existing page an modify it, or you can use builder to create quickly your page. After creating it, you will be able to edit it of course if you want to change small elements, or customize a section.
